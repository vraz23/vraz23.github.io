.NET 9's Secure Foundation

Transitive Dependency Scanning

Automatically finds vulnerabilities deep in your package tree, preventing hidden supply chain attacks.

Pushed Authorization Requests (PAR)

Secures OAuth/OIDC flows by moving sensitive parameters to a secure back-channel, preventing tampering and PII leaks.

ASP0026 Analyzer

Warns developers when an `[Authorize]` attribute is accidentally overridden, preventing unintentional security holes.

The New Frontier of Risk: Adversarial AI

While AI is a powerful defender, the models themselves can be attacked. Securing the AI is as important as securing the application with AI.

🎯 Data Poisoning

Corrupting training data to create backdoors or teach the model to ignore specific threats.

🛡️ Defense: Anomaly Sensors

Apply anomaly detection to the training data itself to identify and remove malicious inputs before training.

🎯 Evasion Attacks

Making tiny, imperceptible changes to input to trick a model into making a wrong decision.

🛡️ Defense: Adversarial Training

Intentionally train the model on adversarial examples to make it more resilient to manipulation.
